# Minecraft 簡繁轉換詞彙表

一個針對 Minecraft 製作的 OpenCC 詞彙表。

藉鑑了 [Blackrowtw](https://github.com/Blackrowtw/Masa-series-mods-translation-zh_tw/tree/master) 的想法，將 [SkyEye FAST 維護的翻譯表](https://github.com/SkyEye-FAST/mc_lang)製作成 OpenCC 的詞彙轉換表。
